SearchEnginePosition is a simple lib created by Fernando Gomes in order to help Seo Job for Ruby programmers.
Basicly it returns a search engine position of a given website, on a given search engine for the given keywords.

References:
Google languages and Local Domain is here: http://www.google.com/language_tools?hl=EN
MSN around the world is here: http://www.msn.com/worldwide.aspx
MSN languages can ben found at the options search: http://search.msn.com/settings.aspx?ru=%2fresults.aspx%3fq%3dbla%26qb%3d1%26FORM%3dAXRE&FORM=SERE (See HTML source code)
Yahoo international sites can be found here: http://world.yahoo.com/
Yahoo languages can be found in Advanced Search, or here: http://search.yahoo.com/web/advanced?ei=UTF-8&p= (See HTML source code)
